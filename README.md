# pmn-flashcards-jp
PMN とは？

PMN（Pivotal Mining Note）は、
ビットコインのマイニングインフラに基づいた金融商品であり、
長期的なビットコインの蓄積を目的としています。

特徴：
	•	実体のあるマイニング基盤
	•	期間固定型のハッシュレート構造
	•	米国の証券規制に準拠した設計

PTS について

Pivotal Trend Service（PTS） は、
実体のあるビットコインマイニングインフラを基盤に、
コンプライアンス重視の金融商品を構築しています。

私たちが重視しているのは：
	•	透明性
	•	長期視点
	•	インフラ起点の設計思想
